# Documentation & Uploads

Use this folder to upload documents, notes, screenshots, and other files from your mobile device.

## Suggested Subfolders

- `notes/` - Quick notes and voice transcriptions
- `screenshots/` - App screenshots, dashboard captures
- `exports/` - Data exports from various tools
- `references/` - Articles, PDFs, reference materials

## Upload via GitHub Mobile

1. Open GitHub app
2. Navigate to this `docs/` folder
3. Tap **+** button (top right)
4. Choose **Upload files** or **Create file**
5. Select files from your phone

## Quick Upload Tips

- Name files descriptively: `2025-01-15-inventory-dashboard.png`
- Add a brief description in the commit message
- Files will sync to all your devices via git
