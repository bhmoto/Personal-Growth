# Claude History

Upload your Claude conversation exports here.

## How to Export from Claude

1. Go to [claude.ai](https://claude.ai)
2. Click your profile icon (bottom left)
3. Select **Settings**
4. Go to **Account** tab
5. Click **Export Data**
6. Download the ZIP file
7. Upload the contents to this folder

## Folder Structure

```
claude-history/
├── README.md (this file)
├── export-YYYY-MM-DD/     <- Create a folder with the export date
│   ├── conversations.json
│   └── ...other files
```

## Upload via Mobile

1. Navigate to this folder on GitHub mobile
2. Tap the **+** button
3. Select **Upload files** or **Create file**
4. Upload your exported JSON files
